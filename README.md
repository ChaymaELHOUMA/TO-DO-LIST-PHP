# To-Do List en PHP

## Description
Ce projet est une **To-Do List simple en PHP**, permettant à l'utilisateur de :

- Ajouter des tâches  
- Marquer une tâche comme terminée ou non terminée  
- Supprimer des tâches  

Les tâches sont sauvegardées dans un fichier `tasks.json`, ce qui permet de conserver la liste même après la fermeture du navigateur.  
Le projet inclut également un **design moderne et stylisé avec CSS**.

---

## Fonctionnalités
- Ajouter une tâche via un formulaire simple  
- Marquer une tâche comme terminée ou non terminée  
- Supprimer une tâche  
- Interface utilisateur responsive et agréable  
- Gestion des caractères spéciaux grâce à `htmlspecialchars`  

---

## Prérequis
- Serveur local compatible PHP (ex : **XAMPP**, **WAMP**, **MAMP**)  
- PHP 7 ou supérieur
